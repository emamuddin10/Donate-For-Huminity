1. What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents like HTML and XML. It represents the document as a hierarchical tree structure where each element, attribute, and piece of content is a node.

2. How do you select an element from the DOM?

To select an element from the DOM, JavaScript provides several methods. These methods allow you to target elements by ID, class, tag name, or using CSS selectors.

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to manage events for its child elements. Instead of adding event listeners to individual child elements, the parent listens for events and determines which child triggered the event using the event object's target property.

4..How do you manipulate an element's attributes and styles using the DOM?

In JavaScript, the DOM provides methods and properties to manipulate the attributes and styles of HTML elements dynamically.
